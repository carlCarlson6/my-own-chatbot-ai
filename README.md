# my-own-chatbot-ai

My own chatbot web app.

## Tech Stack

### Backend
- **.NET** with **Microsoft Orleans** (actor model framework)

### Frontend
- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** for styling
- **Zustand** for state management
- **Zod** for schema validation

### AI
- **Ollama** for LLM inference
